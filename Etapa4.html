Etapa 4

Nesta etapa, iniciamos o desenvolvimento do projeto.

Todo o código será executado em um único programa, o qual será responsável por chamar as funções específicas que permitirão ao usuário realizar diversas ações na plataforma.

Entre as funcionalidades disponíveis, destacam-se:

  • Cadastro de usuários na plataforma;
  • Calibração das câmeras;
  • Identificação do usuário para verificar se a liberação é permitida;
  • Geração de alertas quando um usuário for liberado.
