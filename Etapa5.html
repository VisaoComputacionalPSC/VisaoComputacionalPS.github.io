Sistema de Controle de Acesso com Visão Computacional

O Sistema de Controle de Acesso com Visão Computacional foi desenvolvido para melhorar a segurança em ambientes escolares, como salas de aula, escritórios ou outras áreas restritas. O controle de acesso automatizado é fundamental para garantir que apenas pessoas autorizadas possam acessar determinados locais, oferecendo mais segurança para alunos, professores e colaboradores.

Em muitas instituições ainda se utilizam métodos manuais de controle de entrada, como listas de presença, cartões ou senhas, os quais podem ser facilmente burlados ou esquecidos. Com isso, o sistema proposto utiliza reconhecimento facial para identificar quem está tentando acessar o ambiente, permitindo ou bloqueando o acesso automaticamente. O sistema é baseado em câmeras e inteligência artificial, garantindo maior agilidade e precisão na verificação.

Objetivo do sistema: Fornecer uma solução segura, prática e eficiente para o controle de acesso em locais restritos, utilizando a tecnologia de reconhecimento facial para autorizar ou negar a entrada de forma autônoma, sem a necessidade de operadores humanos, senhas ou cartões.

Como funciona: O sistema é composto por uma câmera que realiza a leitura facial em tempo real, comparando o rosto detectado com um banco de dados de usuários autorizados. Caso o rosto seja reconhecido, a porta é destravada automaticamente, permitindo o acesso. Se o rosto não for identificado, o acesso é negado e um alerta é gerado para o responsável.

Benefícios para o usuário:

  • Segurança Aumentada: bloqueio automático de pessoas não autorizadas.
  • Praticidade: eliminação da necessidade de senhas, chaves ou cartões de identificação.
  • Autonomia: o sistema funciona de forma autônoma, sem intervenção humana.
  • Escalabilidade: o sistema pode ser facilmente adaptado para diferentes ambientes ou unidades.

Procedimento Experimental

Siga os passos abaixo para operar o sistema e coletar os resultados:

  1. Abrir o sistema: Clique no ícone "Iniciar Sistema" na tela principal.
  2. Configuração inicial: Ajuste os controles deslizantes para a intensidade da luz solar e o ângulo dos painéis. A intensidade deve ser ajustada entre 0 a 100% e o ângulo entre 0º e 90º.
  3. Ativar o sistema: Após configurar os parâmetros, clique no botão "Iniciar Operação".
  4. Acompanhe os resultados: Visualize a performance do sistema na tela de resultados, onde gráficos de eficiência energética serão exibidos.
  5. Coleta de dados: Após 5 minutos, o sistema fornecerá uma tabela com a eficiência de conversão de energia e o consumo total. Anote os dados apresentados.
