Introdução

O Sistema de Painéis Solares (SPV) foi desenvolvido para demonstrar o funcionamento de um sistema fotovoltaico através de uma interface gráfica intuitiva. O objetivo deste experimento é possibilitar que o usuário, mesmo sem conhecimento técnico, consiga operar o sistema, visualizar suas funcionalidades e coletar dados relevantes.

O sistema conta com as seguintes interfaces de entrada/saída:

  • Interface de Entrada: Usuário pode configurar parâmetros do sistema como a intensidade da luz solar, ângulo dos painéis e carga do sistema.
  • Interface de Saída: Resultados de eficiência energética, desempenho do sistema e consumo de energia serão apresentados em gráficos e tabelas.

Procedimento Experimental

Siga os passos abaixo para operar o sistema e coletar os resultados:

  1. Abrir o sistema: Clique no ícone "Iniciar Sistema" na tela principal.
  2. Configuração inicial: Ajuste os controles deslizantes para a intensidade da luz solar e o ângulo dos painéis. A intensidade deve ser ajustada entre 0 a 100% e o ângulo entre 0º e 90º.
  3. Ativar o sistema: Após configurar os parâmetros, clique no botão "Iniciar Operação".
  4. Acompanhe os resultados: Visualize a performance do sistema na tela de resultados, onde gráficos de eficiência energética serão exibidos.
  5. Coleta de dados: Após 5 minutos, o sistema fornecerá uma tabela com a eficiência de conversão de energia e o consumo total. Anote os dados apresentados.
