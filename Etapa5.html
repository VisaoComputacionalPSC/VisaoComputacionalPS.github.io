Sistema de Controle de Acesso com Visão Computacional

Introdução

O Sistema de Controle de Acesso com Visão Computacional foi desenvolvido para melhorar a segurança em ambientes escolares, como salas de aula, escritórios ou outras áreas restritas. Ele utiliza tecnologia de reconhecimento facial para garantir que apenas pessoas autorizadas possam acessar locais específicos, proporcionando mais segurança para alunos, professores e colaboradores.

Em muitos ambientes, os sistemas de controle de acesso ainda dependem de métodos tradicionais como cartões de identificação, senhas ou listas de presença, os quais podem ser facilmente esquecidos ou burlados. Esse sistema propõe uma solução mais eficiente e segura, que usa câmeras e inteligência artificial para reconhecer os rostos das pessoas e permitir ou bloquear a entrada automaticamente, sem intervenção humana.

Para um usuário leigo, o funcionamento do sistema é simples e direto:

  • Cadastro de Usuários: O administrador do sistema (pessoa responsável pela configuração inicial) realiza o cadastro de rostos autorizados através de uma interface simples. Esse cadastro pode ser feito enviando fotos ou capturando imagens diretamente do sistema.
  • Entrada no Ambiente: O usuário autorizado se aproxima da porta ou ponto de acesso onde a câmera está instalada.
  • Reconhecimento Facial: A câmera captura a imagem do rosto do usuário e o sistema verifica se o rosto corresponde a um dos cadastrados.
  • Decisão Automática: Se o rosto for reconhecido, a porta é destravada automaticamente, permitindo a entrada. Caso contrário, o acesso é negado e o sistema envia um alerta.

Interface de Entrada: A entrada do sistema ocorre automaticamente quando o usuário se aproxima da câmera. Não é necessário interagir diretamente com a interface de computador ou app.

Interface de Saída: O sistema é visualmente simples para o usuário. Se o acesso for permitido, uma luz verde ou sinal sonoro pode ser emitido, indicando que a porta foi destravada. Se o acesso for negado, uma luz vermelha ou um aviso sonoro será ativado, e o administrador receberá uma notificação em tempo real.

Benefícios do Sistema:

  • Segurança Aumentada: Bloqueia automaticamente o acesso de pessoas não autorizadas.
  • Praticidade: Não é necessário carregar cartões ou lembrar senhas. Basta seu rosto.
  • Autonomia: O sistema opera de maneira totalmente automática, sem a necessidade de operadores humanos ou intervenções manuais.
  • Escalabilidade: O sistema pode ser facilmente expandido para diversos locais ou unidades, mantendo a mesma estrutura básica de funcionamento.

Procedimento Experimental

Siga os passos abaixo para operar o sistema e coletar os resultados:

  1. Abrir o sistema: Clique no ícone "Iniciar Sistema" na tela principal.
  2. Configuração inicial: Ajuste os controles deslizantes para a intensidade da luz solar e o ângulo dos painéis. A intensidade deve ser ajustada entre 0 a 100% e o ângulo entre 0º e 90º.
  3. Ativar o sistema: Após configurar os parâmetros, clique no botão "Iniciar Operação".
  4. Acompanhe os resultados: Visualize a performance do sistema na tela de resultados, onde gráficos de eficiência energética serão exibidos.
  5. Coleta de dados: Após 5 minutos, o sistema fornecerá uma tabela com a eficiência de conversão de energia e o consumo total. Anote os dados apresentados.
